# PLL memory hooks via chunking

On 2026-06-16 Matt reported the two PLL algorithms were "in my memory but not in my hands" and asked for a memory hook for each, since they're significantly longer than anything he'd learned before (A-perm 9 moves, U-perm 11 moves). This is a fluency/storage-strength problem, not a knowledge one — the algorithms were already taught and verified (record 0010). Lesson 13 ("Hooks for the Long Algorithms") was written to solve it.

## The approach: chunking, not new content

The transferable skill taught is **chunking** — holding a long algorithm as a few named bundles + a rhythm, rather than a move string. Grounded in Matt's existing mental model: the trigger `R U R' U'` is already one chunk to him, and the Sune is "trigger + tail." The lesson reuses that frame rather than introducing anything new.

## The two hooks (built from the verified move structure)

- **A-perm `R' F R' B2 R F' R' B2 R2` → "Two Walls."** The two `B2`s are heavy "walls" splitting it into three rooms: `R' F R'` (knock) · `B2` · `R F' R'` (twin knock) · `B2` · `R2` (slam). The two knocks are twins — *flip the first two moves, keep the tail `R'`* — so it compresses to **knock, wall, slam**. Rhythm: "knock-knock-knock — WALL — knock-knock-knock — WALL — SLAM."
- **U-perm `R U' R U R U R U' R' U' R2` → "Two See-Saw Bars."** Only R and U turns. Two identical `R U R U` bars whose U-directions flip between them (bar one dips `U' U`, bar two rises `U U'` — a see-saw), then the landing `R' U' R2` (the only prime/double). Compresses to **bar, bar, land**.
- **Shared shape: "twin + tail"** — both reduce to two near-identical chunks plus a short closer, the same shape as the Sune. This is the real takeaway: hunt for the twin in any future long algorithm.

## Feedback loop — deliberately different from lesson 12

Lesson 12's trainers drill **move-by-move** recall. Lesson 13's trainers drill **chunk-by-chunk** order recall (pad buttons are whole chunks, with one decoy each), training the compressed representation that *is* the hook. Same Guided→Blind, blind-streak-of-5 mechanic. The two lessons are complementary, not redundant.
