# Rubik's Cube Resources

## Knowledge

- [Ruwix — Beginner's Method](https://ruwix.com/the-rubiks-cube/how-to-solve-the-rubiks-cube-beginners-method/)
  The canonical 7-step layer-by-layer guide with exact algorithms and notation. Use for: the backbone of the whole curriculum — every stage of the solve.
- [SolveTheCube.com](https://solvethecube.com/)
  Interactive step-by-step beginner guide with animated moves. Use for: when a written algorithm isn't clicking and Matt needs to see the moves animated.
- [Official "You Can Do The Cube" Solution Guide (PDF)](https://assets.ctfassets.net/r3qu44etwf9a/6kAQCoLmbXXu29TTuArrk1/404118e1f9bfb6f9997157a284bbc572/Rubiks_Solution-Guide_3x3.pdf)
  Rubik's official printable beginner guide. Use for: an offline/printed reference to keep next to the cube.
- [Ryan Heise — Beginner's Solution](https://www.ryanheise.com/cube/beginner.html)
  A more understanding-oriented beginner solution. Use for: the "explain what each stage is doing" success criterion — why the algorithms work, not just what they are.
- [rubikscu.be — Online simulator & solver](https://rubikscu.be/)
  In-browser cube simulator, timer, and solver. Use for: practising notation without scrambling the physical cube; checking a stuck state.
- [J Perm — Beginner 3x3 guide](https://jperm.net/3x3)
  Widely-trusted beginner guide (text + video) that minimises memorisation by reusing the `R U R' U'` trigger for corners. Use for: the corners step and any stage where Ruwix's case-based algorithms feel like too much to memorise. **Chosen as our corners method** — fewest algorithms, fits the mission.

## Knowledge — intermediate (last-layer upgrade, added 2026-06-09)

- [TactileCuber — 2-Look OLL](https://tactilecuber.com/3x3/cfop/2-look-oll/)
  Clean list of all 2-look OLL cases with exact algorithms. Use for: the OLL lessons — verified source for the edge and corner orientation algs.
- [J Perm — 2-Look OLL](https://www.jperm.net/algs/2lookoll)
  Our chosen guide's intermediate path. Use for: the canonical 4LLL framing (orient then permute) and video walkthroughs.
- [Speedsolving Wiki — Corner Orientation](https://www.speedsolving.com/wiki/index.php/Corner_Orientation)
  Reference theory on the corner-orientation cases. Use for: explaining *why* the Sune-repeat works.
- [speedcube.us — Intermediate last layer](https://www.speedcube.us/blogs/speedcubing-solutions/how-to-solve-the-3x3-last-layer-alternate-method-intermediate)
  Describes orienting all corners with only the Sune, including the holding rule. Use for: the minimal-memorisation OLL-corners method (lesson 11).
- [J Perm — 2-Look PLL](https://www.jperm.net/algs/2look/pll) — **chosen primary source** for PLL (lesson 12); JS-rendered so WebFetch can't read it, use the video/page directly. J Perm's "Ua" matches our edge perm exactly.
- [Speedsolving Wiki — PLL](https://www.speedsolving.com/wiki/index.php/PLL) — verified source for the 2-look case structure (3 corner cases Aa/Ab/E, 4 edge cases Ua/Ub/H/Z) and headlights recognition.
  NOTE: the CubeSkills 2-Look PLL PDF (`/uploads/pdf/tutorials/2-look-pll.pdf`) and `/tutorials/2-look-pll` both return 404 as of 2026-06-15 — don't rely on them. Every algorithm used in lesson 12 was verified independently by cube simulation, which is the source of truth going forward.

## Knowledge — F2L (first two layers, added 2026-06-17)

- [J Perm — 3x3 / F2L](https://jperm.net/3x3) — **chosen primary source** for F2L. The most trusted intuitive-F2L tutorial (video). Watch only as far as the basic connected-pair insert for lesson 15.
- [Ruwix — First Two Layers (F2L)](https://ruwix.com/the-rubiks-cube/advanced-cfop-fridrich/first-two-layers-f2l/) — text source with the basic-case move sequences (connected pair → `R U R'`; white-up → setup then insert; split-pair reunion). Use for: the exact moves behind the intuitive cases.
- [speedcube.us — Beginner F2L](https://www.speedcube.us/blogs/speedcubing-solutions/3x3-beginner-f2l-first-two-layers-of-cfop-method) — the three beginner cases (different colours up / same colours up / white up) and the intuitive "pair then insert" framing. Use for: the gentlest case taxonomy.
- [CubeSkills — F2L (Zemdegs/Klise PDF)](https://www.cubeskills.com/uploads/pdf/tutorials/f2l.pdf) — the full 41-case algorithm sheet. Use for: later lessons only, when moving past intuitive pairing. Overkill for now.
